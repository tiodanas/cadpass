Cadastro de Materiais no LIC

O LIC utiliza a mesma base de dados do cadastro de materiais do sistema NUC.
A classificação de materiais no LIC é a mesma do NUC: Grupos, Classes, Materiais Genéricos e Materiais.
A inclusão de novos materiais não pode ser feita no LIC.
